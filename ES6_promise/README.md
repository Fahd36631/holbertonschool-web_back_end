# ES6 Promise

This project covers JavaScript promises and async/await basics.

## Learning Objectives

- Understand promises: why and how to use them
- Use `then`, `catch`, and `resolve`
- Use Promise static methods
- Handle errors with `throw` and `try/catch`
- Use `await` and `async` functions

## Requirements

- Node.js 20.x.x
- npm 9.x.x
- JavaScript files use `.js` extension
- All functions must be exported
- Files end with a new line
