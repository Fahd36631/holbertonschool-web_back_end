# Pagination

This project implements different pagination strategies for a dataset of popular baby names.

## Files

- `0-simple_helper_function.py`: Contains a helper function to calculate pagination index ranges.
- `1-simple_pagination.py`: Implements simple pagination with page and page_size parameters.
- `2-hypermedia_pagination.py`: Implements hypermedia pagination with HATEOAS metadata.
- `3-hypermedia_del_pagination.py`: Implements deletion-resilient hypermedia pagination.

## Requirements

- Python 3.9
- Ubuntu 20.04 LTS
- pycodestyle 2.5.*
